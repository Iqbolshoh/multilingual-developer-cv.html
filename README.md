## 🎯 About The Project

This is an **interactive, multilingual CV** designed for **Iqbolshoh Ilhomjonov** — a Full-Stack Developer from Samarkand, Uzbekistan who speaks **5 languages**. The project showcases his linguistic diversity and technical expertise in a modern, glassmorphic interface.

### Why this project?
- 🌍 Demonstrates multicultural communication skills
- 💼 Highlights professional experience dynamically
- 🎨 Shows modern frontend development abilities
- 🔄 Interactive language switching without page reload

## 🚀 Features

| Feature | Description |
|---------|-------------|
| 🌐 **5 Languages** | English, Russian, Uzbek, Tajik, Persian |
| 💾 **LocalStorage** | Saves preferred language automatically |
| 🎨 **Glassmorphism UI** | Modern blur effects and gradients |
| 📱 **Fully Responsive** | Works on mobile, tablet, desktop |
| 🔘 **Active Button State** | Visual feedback for current language |
| 🍞 **Toast Notifications** | Smooth alerts when language changes |
| 📊 **Dynamic Content** | Bio, skills, experience update in real-time |
| 💼 **Experience Section** | Professional timeline with descriptions |
| 🛠️ **Skills Tags** | Tech stack visualized beautifully |
| 🔗 **Social Links** | GitHub, Telegram, Email, LinkedIn |

## 📊 Content Structure

### Profile Card (Left Column)
- Avatar with emoji
- Name and title (dynamic)
- Location (Samarkand)
- Education (Samarkand State University)
- Experience (3+ years)
- Languages spoken (dynamic)
- Social media links

### Dynamic Content (Right Column)
- **About Me** — Bio in selected language
- **Experience** — Work history with dates
- **Tech Skills** — Technology stack tags
